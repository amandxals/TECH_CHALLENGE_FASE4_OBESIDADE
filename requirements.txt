## 🚀 Como rodar — passo a passo (para leigos)

### Pré-requisito: ter Python instalado
Acesse https://python.org/downloads e instale a versão mais recente.

---

### Passo 1 — Abra o Terminal (ou Prompt de Comando)
- **Windows:** pressione `Win + R`, digite `cmd`, Enter
- **Mac:** pressione `Cmd + Space`, digite `Terminal`, Enter

---

### Passo 2 — Vá até a pasta do projeto
No terminal, navegue até onde você salvou os arquivos.
Exemplo: se salvou em `Downloads/obesity_project`:
```
cd Downloads/obesity_project
```

---

### Passo 3 — Instale as bibliotecas necessárias
```
pip install -r requirements.txt
```
*(isso pode demorar 1-2 minutos)*

---

### Passo 4 — Coloque o arquivo Obesity.csv na pasta
Copie o arquivo `Obesity.csv` para dentro da pasta `obesity_project`.

---

### Passo 5 — Treine o modelo (só precisa fazer UMA vez)
```
python train.py
```
Você verá as acurácias de cada modelo. O melhor é salvo automaticamente.

---

### Passo 6 — Rode o aplicativo!
```
streamlit run app.py
```
Uma janela do navegador abrirá automaticamente com o sistema.
Se não abrir, acesse: **http://localhost:8501**

---

## ☁️ Deploy no Streamlit Cloud (para compartilhar o link)

1. Crie uma conta em https://share.streamlit.io (gratuito)
2. Faça upload do projeto para o GitHub
3. No Streamlit Cloud: clique em "New app" → selecione seu repositório → selecione `app.py`
4. Clique em Deploy — em poucos minutos você terá um link público!

## 🧠 Como o modelo funciona (para leigos)

Imagine que você quer ensinar um computador a reconhecer tipos de peso corporal.
Você mostra para ele 2.111 casos reais de pacientes, com informações como:
- altura, peso, idade, gênero
- se tem histórico familiar de obesidade
- quantas vezes faz exercício, o que come, etc.

O modelo aprende os padrões e, quando chega um paciente novo, ele consegue
prever o nível de obesidade com **95% de precisão**.

O algoritmo usado é o **Random Forest** (Floresta Aleatória):
> Imagine 200 "médicos virtuais" (árvores de decisão) cada um analisando o
> paciente de uma perspectiva diferente. A resposta final é a que a maioria
> dos médicos concorda — isso é muito mais confiável do que um só médico decidir.
